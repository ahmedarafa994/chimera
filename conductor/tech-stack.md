# Technology Stack

## Backend
- **Language:** Python 3.11+
- **Framework:** FastAPI
- **ORM:** SQLAlchemy with Alembic for migrations
- **Data Validation:** Pydantic
- **Caching/Task Queue:** Redis

## Frontend
- **Language/Environment:** Node.js 18+
- **Framework:** Next.js (React)

## AI/ML & Security Research
- **Providers:** Google Gemini, OpenAI, Anthropic, DeepSeek
- **Libraries:** Spacy, PyTorch, Transformers, sentence-transformers
- **Security Frameworks:** DeepTeam, DeepEval

## Infrastructure & Data Engineering
- **Containerization:** Docker & Docker Compose
- **Orchestration:** Airflow (for data pipelines)
- **Data Transformation:** dbt
- **Database:** SQLite (dev) / PostgreSQL (prod inferred from SQLAlchemy usage)

## Quality Assurance & Tools
- **Testing:** Pytest (Backend), Vitest (Frontend), Playwright (E2E)
- **Linting/Formatting:** Ruff, Black, Isort, Biome
- **Security Scanning:** Bandit, Pip-audit, Safety
